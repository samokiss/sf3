Interfaces

trait FooTrait
{
    private function foo()
    {
        return 'Hello';
    }
}

{
    use FooTrait;

    public function bar();
}

class Main implements BarInterface
{
    public function bar()
    {
        return 'World!';
    }

    public function main()
    {
        return $this->foo().' '.$this->bar();
    }
}

$main = new Main();
echo $main->main();

What will be the outcome of executing this code snippet?

  • PHP will raise a Fatal Error telling traits are not allowed inside interfaces.

The choices were

  • The script will successfully run and output the string `Hello World!*.
