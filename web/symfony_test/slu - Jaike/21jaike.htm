The "with" tag

In the following Twig template:

Twig
{% set maxItems = 10 %}

{% with { maxItems: 5 } %}
    {% for i in 1..maxItems %}
        {# ... #}
    {% endfor %}
{% endwith %}

{% for j in 1..maxItems %}
    {# ... #}
{% endfor %}

Which will be the last values iterated by i and j variables in the for loops?

Correct

  • i = 5 and j = 10

The choices were

  • i = 5 and j = 10

  • i = 10 and j = 5

  • i = 10 and j = 10

  • i = 5 and j = 5

Help

Twig 1.28 (November 2016) introduced a new with tag that allows to create different scopes in the same template. Variables defined inside a with tag are not available outside of it and don't modify the external variables. See http://twig.sensiolabs.org/doc/tags/with.html
