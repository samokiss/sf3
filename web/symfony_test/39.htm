Single choice

Checking for blocks

Consider the following Twig snippet:

Twig
{% if block('footer', 'common_blocks.html.twig') is defined %}
    ...
{% endif %}

Which of the following statements are true?

Correct

  • The code checks if the common_blocks.html.twig template contains a Twig block called footer

The choices were

  • The code checks if the common_blocks.html.twig template contains a Twig block called footer

  • The code is wrong because the block() function doesn't allow to pass a second argument.

  • The if condition will be false if the footer block exists in the common_blocks.html.twig template but it's empty (it doesn't have any content inside).

  • The code is wrong because the is defined test cannot be used with the block() function.

Help

Twig 1.28 (November 2016) added support for the is defined operator and the second argument of the block() function, allowing to check if a given Twig block exists in a template. See http://twig.sensiolabs.org/doc/functions/block.html
