Array counting

What is the most recommended way to count the number of occurences of each unique value in the following array?

PHP
$a = [1, 1, 2, 3, 4, 4, 5, 6, 6, 6, 6, 3, 2, 2, 2];

Correct

  • PHP
    $n = array_count_values($a);

The choices were

  • PHP
    $k = array_unique($a);
$v = array_fill(0, count($k), 0);
$n = array_combine($k, $v);

foreach ($a as $i) {
    ++$n[$i];
}
  • PHP
    $n = array_total_values($a);
  • PHP
    $k = array_unique($a);
$v = array_fill(0, count($k), 0);
$n = array_combine($k, $v);

reset($a);
for ($i = 0; $i < count($a); $i++) {
    ++$n[current($a)];
    next($a);
}
  • PHP
    $n = array_count_values($a);
  • PHP
    $n = coun($a);

Help

  • http://www.php.net/arrays
